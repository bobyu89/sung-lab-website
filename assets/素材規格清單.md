# 素材規格清單

本清單列出網站目前使用「佔位圖」或「文字替代」的位置，說明未來要換成真實素材時：
**檔名該取什麼、放到哪個資料夾、需不需要動到程式碼（以及動哪一行）**。

網站目前所有頁面在沒有真實素材時都會自動顯示佔位圖或漸層插畫，不會破版、不會出現「圖片壞掉」的裂圖示，可以放心之後再慢慢補素材。

---

## 1. 首頁大圖 hero-illustration.png（優先度：高）

| 項目 | 規格 |
|---|---|
| 檔名 | `assets/hero-illustration.png` |
| 放置位置 | 專案根目錄的 `assets/` 資料夾（與 8 張 `placeholder-*.svg` 同層） |
| 比例 | **4:3**（寬:高，例如 1600×1200px、1200×900px 皆可） |
| 建議最小寬度 | ≥ 1200px（確保在大螢幕不糊） |
| 背景 | 去背（透明 PNG）或白色背景皆可，**不要**用粉色/橘色背景（會跟頁面底色疊在一起分不清楚） |
| 檔案格式 | PNG（若能去背最好；也接受 JPG 但無法去背） |
| 需要動到程式碼嗎 | **不需要**。`index.html` 第 39 行已經寫好 `<img src="assets/hero-illustration.png">`，檔名、路徑完全對應，只要把檔案放進 `assets/` 資料夾，重新整理網頁就會自動出現。目前因為檔案不存在，會自動顯示一個粉橘漸層愛心圖示佔位（`index.html:40` 的 `onerror` 屬性），這是正常設計，不是壞掉。 |

### 風格描述

網站的視覺主色是「粉紅 `#FF79BC` → 橘 `#FFA042`」漸層，整體走**扁平插畫（flat illustration）**風格：線條乾淨、色塊為主、不寫實、帶一點溫暖親切感，符合「護理／照護／科技創新」的主題。建議插畫內容：一位護理人員或研究者的形象，搭配聽診器、平板電腦、3D 列印模型、老年關懷或数据圖表等元素之一，整體構圖偏向溫暖、專業、有希望感，避免冷冰冰的醫療器材特寫。

## 2. 各分頁頁首選配圖 page-hero（優先度：低，可省略）

目前 `research.html`、`education.html`、`members.html`、`collaboration.html`、`resources.html` 的頁首（`.page-hero` 區塊）**只有文字**（標題＋一句話說明），沒有配圖，版面本身已經設計完整，**不補圖也完全沒問題**。

如果之後想幫每個分頁加一張示意圖：

| 項目 | 規格 |
|---|---|
| 建議檔名 | `assets/page-hero-研究主題.png`、`assets/page-hero-教學資源.png`……依頁面命名，用中文頁名即可 |
| 放置位置 | `assets/` 資料夾 |
| 比例 | 建議 16:9 或 3:1 橫幅，寬度 ≥ 1200px |
| 需要動到程式碼嗎 | **需要**。目前 `.page-hero` 區塊沒有 `<img>` 標籤，要新增圖片需請工程人員在對應頁面的 `<div class="page-hero__inner">` 內加一行 `<img>`，並在 `css/main.css` 補上對應樣式（屬於小型結構調整，非單純換檔）。**這一項屬於「加分」而非必要項目，建議暫緩，優先處理第 1、3 項。** |

## 3. 成員照片（優先度：中，等成員名單確定後處理）

| 項目 | 規格 |
|---|---|
| 建議檔名 | `assets/members/宋建美.jpg`（之後其他成員比照：`assets/members/姓名.jpg`） |
| 放置位置 | 建議新增 `assets/members/` 子資料夾集中存放 |
| 比例 | **1:1 正方形** |
| 建議最小尺寸 | ≥ 400×400px |
| 檔案格式 | JPG 或 PNG 皆可 |
| 需要動到程式碼嗎 | **需要**。目前 `members.html` 第 59 行主持人頭像是文字圓形色塊 `<span class="avatar member-intro__avatar">宋</span>`（顯示姓氏「宋」字），**不是** `<img>` 標籤。要換成真實照片，需請工程人員將這一行改成 `<img class="avatar member-intro__avatar" src="assets/members/宋建美.jpg" alt="宋建美 助理教授">`（**兩個 class 都要保留**：`avatar` 提供圓形裁切 `border-radius: 50%`，`member-intro__avatar` 提供 120×120px 的大尺寸；換成 `<img>` 後圖片會自動使用 `css/main.css` 裡的 `img.avatar { object-fit: cover; }` 規則裁切填滿圓形，不會變形，工程人員不需要再另外補 CSS）。**建議把這件事和「成員名單整理中」佔位一起，等老師確認完整團隊名單與每人照片後，一次交給工程人員處理。** |

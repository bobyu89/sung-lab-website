## 1. 目標

為國防醫學大學護理學院宋建美助理教授建置多頁式研究室網站：粉色溫暖護理風、五層內容架構（品牌／學術／教育／合作／資源）、老師可透過 Google 試算表與雲端硬碟自行更新消息與照片。

### 品牌定調
- 研究室名稱：**高齡照護與護理創新研究室**／Geriatric Care & Nursing Innovation Lab（名稱集中於 `js/data.js` 一處設定，日後可換）
- 定位一句話（草案，實作時可潤）：結合高齡照護、實證研究、教育創新與科技應用，以證據推動照護品質與健康福祉。

### 非目標（這版不做）
- 中英全站切換、後台管理介面、部落格系統
- 由 Claude 手繪 SVG 人物插畫（使用者明確否決；用可替換圖片插槽）

## 2. 技術方案

- 純靜態 HTML + CSS + 原生 JS，無框架、無建置工具
- 部署：Firebase Hosting（**需為本站新建 Firebase 專案**，不可用賀老師的；佈署步驟寫入維護說明）
- 共用頁首頁尾：`js/components.js` 模板注入
- 最新消息：公開發布的 Google 試算表（gviz JSON 端點）
- 活動照片：Google 雲端資料夾（Drive API v3 + API 金鑰，縮圖 `drive.google.com/thumbnail?id=...&sz=w800`）
- 降級策略:任一外部資料源失敗顯示內建備用內容，版面不破
- 聯絡表單：mailto（組主旨內文開郵件軟體，寄 sungcm@mail.ndmutsgh.edu.tw）

## 3. 視覺系統（依範例圖）

### 配色

| 角色 | 色票 | 用途 |
|---|---|---|
| 主色 | #E85D8A 玫瑰粉 | 標題強調、圖示、裝飾 |
| 主色深 | #D6336C 深粉 | 連結、需對比的粉色文字 |
| 輔色 | #F59B53 珊瑚橘 | CTA「加入我們」、次要強調 |
| 底色 | #FFFFFF 白 | 頁面基底 |
| 區塊底 | #FDEEF2 淺粉 | 交錯區塊背景、Hero 基調 |
| 卡片變化 | #FFF6E8 奶油黃 | 部分卡片底 |
| 文字 | #3D3A3A 暖炭灰 | 標題、內文 |

文字一律 #3D3A3A 或 #D6336C，達 WCAG AA；淺粉/珊瑚橘僅作背景、按鈕底（配白字需驗對比）與裝飾。

### 造型語言
大圓角卡片（16–24px）、膠囊按鈕、粉色柔和陰影、愛心＋十字 logo、點陣 pattern、心電圖裝飾線、柔和 blob；字體 Noto Sans TC（400/500/700/900）＋Inter 英文小標。

### 圖片素材策略
- 基底不依賴人物插畫：漸層、inline SVG 圖示、心電圖線撐起風格
- Hero 與各區大圖＝規格化 `<img>` 插槽＋粉色漸層佔位圖
- 交付 `assets/素材規格清單.md`：每插槽尺寸、比例、建議 AI 生圖 prompt，換檔即替換
- 主持人與成員照片：先用姓名縮寫圓形頭像佔位

### 動畫（溫和，全部支援 prefers-reduced-motion）
捲動進場淡入上滑（錯開 0.1s）、卡片/按鈕 hover 浮起、統計數字滾動、心電圖線緩慢流動、照片牆 hover＋燈箱。

## 6. Google 後台資料架構

### 試算表（一份、兩工作表、發布到網路）
- **news**：`date`（YYYY-MM-DD）、`category`（榮譽｜發表｜活動｜公告）、`content`、`link`（選填）
- **events**：`date`、`title`、`description`、`image`（選填）、`pinned`

讀取：`https://docs.google.com/spreadsheets/d/{ID}/gviz/tq?tqx=out:json&sheet={name}`，日期新→舊。

### 雲端資料夾（活動照片）
Drive API v3 `files.list`（API 金鑰由使用者建立、限制 referrer）；檔名＝照片說明；合作交流頁列全部；未設金鑰時用備用佔位照片。

### 備用資料
`js/data.js` 內建備用消息（可用真實事實：2026 IJNS 論文刊出、2024 護理雜誌中文論文等）與佔位照片。

### 交付附「維護說明」
README 教老師：試算表加消息、照片上傳與命名、素材規格清單用法、Firebase 專案建立與 `firebase deploy`。

## 8. 測試與驗收

- 桌機（1280+）／平板（768）／手機（375）三尺寸逐頁檢查（Preview 工具實測）
- 外部資料源斷線測試：試算表/Drive 失敗 → 備用內容正常顯示
- `prefers-reduced-motion` 開啟時動畫停用
- Lighthouse：效能與無障礙 ≥ 90
- 內容比對 `docs/content/pi-sung-chien-mei.md`：學經歷、專長、11 計畫、21 篇已擷取論文零遺漏；論文頁建置時再對照來源頁補齊至 23 篇
- 佔位區塊呈現「建置中」樣式而非空白

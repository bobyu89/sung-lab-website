# 護理創新及專科訓練研究室（宋建美老師）— 網站設計文件

日期：2026-07-06（同日更新：套用使用者提供之品牌標誌組合與風格指南）
狀態：已定案
內容來源：`docs/content/pi-sung-chien-mei.md`（擷取自 https://wwwndmc.ndmutsgh.edu.tw/DocDet/191/100010/1738/4100 ）
視覺基準：使用者提供之粉色護理研究室範例圖＋**品牌套件四張圖**（logo kit／hero 插畫／12 款 icon pack／裝飾元素與風格指南）

本專案獨立於 ycho-lab-website（賀老師網站）；兩者互不相干。

## 1. 目標

為國防醫學大學護理學院宋建美助理教授建置多頁式研究室網站：粉色溫暖護理風、五層內容架構（品牌／學術／教育／合作／資源）、老師可透過 Google 試算表與雲端硬碟自行更新消息與照片。

### 品牌定調（使用者 2026-07-06 提供）
- 研究室名稱：**護理創新及專科訓練研究室**；英文暫定 **Nursing Innovation & NP Training Lab**（英文名待老師確認，集中於 `js/data.js` 一處設定）
- 定位標語：**結合專師照護、實證研究、教育創新與科技應用**（＋延伸句：以實證推動照護品質，提升健康福祉）
- Hero 主標（採風格指南示範句型）：「以護理、研究與創新，打造更智慧的臨床未來」——「智慧的臨床」可依老師偏好改「專業的照護」
- 品牌關鍵字：實證・創新・溫度・專業・合作
- Hero 特色籤（採裝飾元素圖）：實證為本 × 創新驅動 × 以人為本

### 非目標（這版不做）
- 中英全站切換、後台管理介面、部落格系統
- 由 Claude 手繪人物插畫（hero 插畫由使用者以 AI 生圖提供）

## 2. 技術方案

- 純靜態 HTML + CSS + 原生 JS，無框架、無建置工具
- 部署：Firebase Hosting（需為本站新建 Firebase 專案；佈署步驟寫入維護說明）
- 共用頁首頁尾：`js/components.js` 模板注入
- 最新消息：公開發布的 Google 試算表（gviz JSON 端點）
- 活動照片：Google 雲端資料夾（Drive API v3 + API 金鑰，縮圖 `drive.google.com/thumbnail?id=...&sz=w800`）
- 降級策略：任一外部資料源失敗顯示內建備用內容，版面不破
- 聯絡表單：mailto（組主旨內文開郵件軟體，寄 sungcm@mail.ndmutsgh.edu.tw）

## 3. 視覺系統（依品牌風格指南）

### 配色（品牌色票，精確值）

| 角色 | 色票 | 語意 | 用途 |
|---|---|---|---|
| 主色 | #FF79BC 莓粉 | 溫暖、關懷、專業 | logo、圖示、裝飾、大型標題強調字 |
| 主色深 | #D9367F（由莓粉加深） | — | 內文連結、小字級粉色文字（#FF79BC 對比不足，不用於小字） |
| 輔色 | #FFA042 橘橙 | 活力、創新、希望 | CTA「加入我們」、次要按鈕描邊、強調字 |
| 輔色底 | #FFE6D6 蜜桃 | 柔和、親和、安心 | 卡片變化底、籤底 |
| 區塊底 | #FFF2F6 淡粉 | 柔和背景、輕盈 | 交錯區塊背景、Hero 基調 |
| 底色 | #FFFFFF 白 | 純淨、簡潔 | 頁面基底、卡片 |
| 中性色 | #6B7280 暖灰 | 穩重、清晰、易讀 | 次要文字 |
| 主文字 | #374151（暖灰加深） | — | 標題、內文（#6B7280 留給輔助文字） |

無障礙：小字級文字一律 #374151／#6B7280／#D9367F；#FF79BC 與 #FFA042 僅用於大標強調、圖示、裝飾與按鈕底（按鈕白字需驗對比，必要時加深底色）。

### Logo（依 logo kit）
- 圖形：四瓣醫療十字＋中央愛心，莓粉→橘橙漸層；以 inline SVG 重建（幾何造型，可精確重現）
- 組合：頁首用橫式（icon＋中文名＋英文小字）；頁尾可用堆疊式或單色淺色版；favicon 用圓角方形 app icon 版
- 安全空間：四周留白 ≥ 圖示高度；最小數位尺寸 24px

### 圖示策略（依 icon pack）
- 12 款主題圖示（臨床護理、智慧醫療/AI、護理教育、實證研究、學術發表、研究專案、學生培育、跨域合作、聯絡我們、心電圖、數據分析、創新研發）對應各區塊
- 實作用開源線性圓角圖示庫（Tabler／Lucide，MIT 授權）上莓粉/橘橙雙色，貼近 icon pack 的 rounded line 風格；若使用者提供 icon pack 原始檔（去背 PNG/SVG）則直接替換

### 裝飾元素（依裝飾元素圖）
心電圖線（區塊分隔）、AI 節點網絡、醫療加號、點陣圖案、漸層 blob、柔和波紋線、徽章籤（進行中／已完成／最新研究等）——以 CSS/SVG 實作，低飽和、放背景角落

### 造型語言
大圓角卡片（16–24px）、膠囊按鈕（主按鈕莓粉底白字＋箭頭、次按鈕橘橙描邊）、粉色柔和陰影；字體 Noto Sans TC（400/500/700/900）＋Inter（英文標題 SemiBold）

### 圖片素材
- **Hero 插畫**：使用者已以 AI 產出（護理師＋筆電＋數據面板），需將原檔存入 `assets/hero-illustration.png`；未提供前用淡粉漸層佔位
- 其他插槽同前：規格化 `<img>`＋佔位圖＋`assets/素材規格清單.md`
- 主持人與成員照片：姓名縮寫圓形頭像佔位

### 動畫（溫和，全部支援 prefers-reduced-motion）
捲動進場淡入上滑（錯開 0.1s）、卡片/按鈕 hover 浮起、統計數字滾動、心電圖線緩慢流動、照片牆 hover＋燈箱

## 6. Google 後台資料架構

### 試算表（一份、一工作表、發布到網路）
- **news**：`date`（YYYY-MM-DD）、`category`（榮譽｜發表｜活動｜公告）、`content`、`link`（選填）

讀取：`https://docs.google.com/spreadsheets/d/{ID}/gviz/tq?tqx=out:json&sheet={name}`，日期新→舊。

### 雲端資料夾（活動照片）
Drive API v3 `files.list`（API 金鑰由使用者建立、限制 referrer）；檔名＝照片說明；合作交流頁列全部；未設金鑰時用備用佔位照片。

### 備用資料
`js/data.js` 內建備用消息（用真實事實：2026 IJNS 論文刊出、2024 護理雜誌中文論文等）與佔位照片。

### 交付附「維護說明」
README 教老師：試算表加消息、照片上傳與命名、素材規格清單用法、Firebase 專案建立與 `firebase deploy`。

## 8. 測試與驗收

- 桌機（1280+）／平板（768）／手機（375）三尺寸逐頁檢查（Preview 工具實測）
- 外部資料源斷線測試：試算表/Drive 失敗 → 備用內容正常顯示
- `prefers-reduced-motion` 開啟時動畫停用
- Lighthouse：效能與無障礙 ≥ 90
- 內容比對 `docs/content/pi-sung-chien-mei.md`：學經歷、專長、11 計畫、21 篇已擷取論文零遺漏；論文頁建置時再對照來源頁補齊至 23 篇
- 佔位區塊呈現「建置中」樣式而非空白
- 品牌色使用符合本文件無障礙規範（小字不得用 #FF79BC/#FFA042）
